# DIO_projeto_ETL
## Projeto desenvolvido no curso de Fundamentos de ETL com Python da Digital Innovation One

No arquivo [*extracao.ipynb*](https://github.com/lilacostaro/DIO_projrto_ETL/blob/main/extracao.ipynb) foi realizada a parte de extração dos dados, que tem como fonte, o arquivo [**ocorrencia.csv**](https://github.com/lilacostaro/DIO_projrto_ETL/blob/main/ocorrencia.csv), do conjunto de dados, "Ocorrências Aeronáuticas na Aviação Civil Brasileira", fornecidos pela **CENIPA**.

No arquivo [*limpeza.ipynb*](https://github.com/lilacostaro/DIO_projrto_ETL/blob/main/extracao.ipynb) foram realizados alguns comandos para uma base de como realizar a limpeza desses dados, estabelescendo um padrão para os valores inexistentes. O mesmo pode ser feito com valores que destoam do padrão, mas essa decisão deve ser avaliada em cada caso, de acordo com os requisitos do projeto.

No arquivo [*transformacao.ipynb*](https://github.com/lilacostaro/DIO_projrto_ETL/blob/main/transformacao.ipynb) foram vistos varios comandos para manipulação dos dados do dataframe, para apartir dele gerar novos dataframes, apenas com as informaçoes que serão analisadas para o projeto em questão. 

A parte de **Load** não foi realizada, uma vez que o intuito do projeto era focar nas primeiras fases do ETL.
